# openclidev.github.io

This repository supports the static files required to implement [openclidev.github.io](https://openclidev.github.io) and subsequently [opencli.dev](https://opencli.dev).

If you are looking for documentation on the OpenCLI Specification standard see [opencli.dev](https://opencli.dev) or the primary OpenCLI repository: https://github.com/bcdxn/opencli
